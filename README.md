Stripe Payments
====================================
Replace "Stripe Publishable Key" & "Stripe Secret Key" in index.php if you want to test it.

Features
===============
* Form validation
* Generate Token via Stripe JS
* Responsive design
* Warning if JavaScript is disabled
* Charge single amounts with Stripe
